Аудиоредактор
Вход: файл(ы) в форматах .wav, .mp3
Реализовать утилиту для редактирования аудиофайлов с возможностью ускорения, замедления, разреза, склейки и т.п.

запуск из консоли:
cd "путь к папке Audio"
python main.py "путь к аудио"
далее вводятся команды
